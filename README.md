# Ecoli-Bionformatics-Introduction
Guided tutorial implementation of nextflow, docker, and bash scripting to create a reproducable bioinformatics pipeline. End-to-end process involving downloading files, FASTQC, trimming, BWA, variant calling and data visualization. Ran within a linux WSL2 environment.

# Reflection

Learned the value of docker containers and which versions to use depending on the current process implemented.
Debugged numerous times process input and ouput (and by extension, publishDir) to seamlessly work between pipeline steps
